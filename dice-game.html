Project Detail

This interactive game ensures a different winner every time the page reloads, adding an element of surprise and excitement to the user experience.

Key Features:
  • Random Dice Rolling: Simulate player turns with random dice rolls using JavaScript.
  • Player Scores: Display and update scores for both players on the webpage.
  • Winner Announcement: Dynamically announce the winner based on the dice rolls.
  • Page Reloading: The game restarts and selects a new winner each time the page is reloaded.
  • Engaging Design: Create an attractive and interactive user interface.
